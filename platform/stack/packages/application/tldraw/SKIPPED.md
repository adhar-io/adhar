# tldraw -- SKIPPED

**Status:** SKIPPED (no official Helm chart, no official self-host image)

tldraw is published as an SDK/library and a hosted product; it does not provide an
official Helm chart or an official self-hosting container image. The available
self-host options are all community Docker projects (e.g. `FoxxMD/tldraw-selfhosted`,
`amirandalibi/tldraw-docker`, and assorted `*/tldraw` Docker Hub images), and the
multiplayer sync server additionally requires a tldraw license. There is no
maintained chart and no canonical upstream image.

Per the platform rules we do not fabricate a chart or invent images, so this
package is intentionally left as a stub.
