# webstudio -- SKIPPED

**Status:** SKIPPED (no official Helm chart, no documented self-host manifest)

Webstudio (webstudio.is) is primarily offered as a hosted SaaS / open-source
builder; it does not publish an official Helm chart, and there is no documented,
maintained Kubernetes self-hosting path or canonical container image to template
from. A web search returned no official Helm chart or self-host deployment docs.

Per the platform rules we do not fabricate a chart or invent images, so this
package is intentionally left as a stub. Revisit if/when upstream publishes an
official chart or a supported container image + manifests.
